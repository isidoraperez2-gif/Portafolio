# Portafolio
Portafolio Ciencia de Datos para la Economia 
## Objetivo del proyecto 
El objetivo de este proyecto es desarrollar un proceso de análisis y modelación de datos orientado a la **predicción del nivel de PIB de distintos países**, utilizando indicadores económicos, sociales y demográficos obtenidos desde el Banco Mundial. 

Para ello, se construye un flujo de trabajo en ciencia de datos que permite preparar, transformar y sintetizar la información disponible, con el fin de generar una variable objetivo adecuada para su posterior clasificación y análisis predictivo.
